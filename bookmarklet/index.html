🔖 Bookmarklet SIASN — KGB App

Tool otomatis untuk mengambil data PNS dari SIASN BKN ke form SK Fungsional

Alur penggunaan:

Isi NIP di KGB App → Klik "Cek SIASN" → Tab SIASN terbuka → Klik Bookmarklet → Data otomatis terisi ✓
1

Aktifkan Bookmark Bar

Pastikan bookmark bar browser Anda terlihat. Di Chrome: Ctrl+Shift+B. Di Edge: Ctrl+Shift+B. Di Firefox: Ctrl+Shift+B.

2

Drag tombol ini ke Bookmark Bar

Klik dan seret tombol ungu di bawah ke bookmark bar browser Anda:

🔖 SIASN → KGB App
⬆ Drag ke bookmark bar, atau klik kanan → "Tambahkan ke Bookmark"

Alternatif: salin URL-nya, buat bookmark baru, paste di field URL/Address.

3

Cara pakai

Di KGB App, isi NIP pegawai → klik tombol "Cek SIASN" → halaman SIASN terbuka → klik bookmark "SIASN → KGB App" → data otomatis terisi ke form.

⚠️ Catatan Penting Data yang diambil: Nama, Golongan Ruang, TMT Golongan, Unit Kerja, Perangkat Daerah. Bookmarklet hanya bekerja jika halaman SIASN sudah menampilkan data pegawai (setelah Cari berhasil). Jika window.opener tidak tersedia, data akan disalin ke clipboard.
